# app_typescript_teste

## Comandos

### Gerar package.json

- npm init

### Pacotes

- npm install typescript
- npm install ts-node
- npm install ts-node-dev
- npm install nodemon

### Gerar tsconfig.json

- npx tsc --init

### Iniciar aplicação

- npm start
- npm run dev
- npm run nodemon_ts
- npm run nodemon_ts_dev
